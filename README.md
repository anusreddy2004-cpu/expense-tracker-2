# Expense Tracker Web Application

A Python Full Stack web application used to manage daily expenses efficiently.

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite
- Chart.js
- Git & GitHub

## Features

- Add Expense
- Edit Expense
- Delete Expense
- Search Expense
- Total Expense Calculation
- Expense Charts
- Login/Register
- Responsive Dashboard UI

## Installation

```bash
pip install flask
python app.py
```
